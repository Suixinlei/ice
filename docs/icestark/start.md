## 使用 CLI 方式创建项目

### 1. 安装 CLI 工具

```bash
$ npm i -g iceworks
```

### 2. 初始化项目

```bash
$ mkdir icestark-layout && cd icestark-layout
$ iceworks init @icedesign/stark-layout-scaffold
```

## 使用 GUI 方式创建项目

### 1. 安装 GUI 工具

```bash
$ npm i -g iceworks
```

### 2. 启动 iceworks

安装完成后，只需要在命令行执行以下命令，即可在浏览器启动本地化 Web 版本：

```bash
$ iceworks
```
